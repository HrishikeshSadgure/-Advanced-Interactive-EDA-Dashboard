# 📊 Interactive EDA Dashboard  
**Upload → Explore → Visualize → Download**

An **interactive Exploratory Data Analysis (EDA) dashboard** that allows users to upload CSV datasets and instantly explore them using **dynamic filters, summaries, and advanced visualizations** — all in a clean, user-friendly interface.

This project is designed to simplify **data understanding, pattern discovery, and insight generation** without writing code.

## ✨ Key Features

✅ Upload CSV files (up to **200MB**)  
✅ Automatic **dataset preview & summary**  
✅ Missing value detection  
✅ Dynamic **column-based filters**  
✅ Download filtered datasets  
✅ **Light / Dark mode toggle**  
✅ Rich **interactive visualizations**

---

## 📂 Dataset Capabilities

Once a CSV file is uploaded, the dashboard automatically displays:

- 📌 **Total Rows**
- 📌 **Total Columns**
- 📌 **Missing Values Count**
- 📌 **Scrollable Dataset Preview**

Users can filter data by:
- Numeric ranges (e.g. Year, Price)
- Column-based selections

---

## 📈 Visualizations Included

The dashboard supports a wide range of visual analytics:

### 🔹 Basic Charts
- Histogram  
- Box Plot  
- Line Chart  
- Bar Chart  
- Scatter Plot  

### 🔹 Advanced Visualizations
- 📊 **Correlation Heatmap**
- 🎻 **Violin Plot**
- 🌳 **Tree Map**
- 🥧 **Pie Chart**
- 🔗 **Pair Plot**
- 🧊 **3D Scatter Visualization**

Each visualization updates **dynamically** based on selected filters and columns.
